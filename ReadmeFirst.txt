After downloading the code you will need following things to run the frontend on local server:
1-->Node JS must be Installed in  your computer.
2-->after downloading code select the directory '..../FYP/FYP-frontend' and type 'npm install' in your terminal to install all
dependencies needed to run React JS.
3-->In the same directory '.../FYP/FYP-frontend' type 'npm run start' in your terminal to start your project running on
localhost.
